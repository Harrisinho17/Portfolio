AI/ML Football Analysis

Overview

Developed by the Youtuber of “Code in a Jiffy”, this project harnesses the power of machine learning, computer vision, and deep learning to create a comprehensive football analysis system. Utilizing YOLO for object detection and custom training, the system can identify and track players, referees, and footballs. It employs KMeans clustering for player segmentation and optical flow to track camera movements. Additionally, CV2’s perspective transformation tools measure scene depth, enabling precise calculations of player movement in meters. This project serves as a practical guide for both beginners and advanced learners to apply these technologies in real-world scenarios, significantly enhancing sports analytics capabilities.
